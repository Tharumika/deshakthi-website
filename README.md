## 📖 About

Professional company website for **Deshakthee Lanka Agencies (Pvt) Ltd.**, a trusted leader in recruitment and manpower supply since 1998. The company specializes in connecting Sri Lankan human resources with global employers across the Middle East, Singapore, Malaysia, Cyprus, and the Maldives.

🏆 **Award-winning agency** — recognized with Excellence Awards in 2012 and 2014 by the Sri Lanka Bureau of Foreign Employment.

## ✨ Features

- 🎨 **Professional Design** — Clean, modern UI with logo-derived color palette (Green + Gold)
- 📱 **Fully Responsive** — Optimized for desktop, tablet, and mobile (375px → 1440px)
- ⚡ **Fast Loading** — Zero framework overhead, pure HTML/CSS/JS
- 🔍 **SEO Optimized** — Semantic HTML5, meta tags, Open Graph support
- 🎬 **Scroll Animations** — Smooth reveal effects powered by AOS library
- ⌨️ **Typing Effect** — Dynamic hero headline with rotating text
- 📊 **Animated Counters** — Statistics that count up on scroll
- ♿ **Accessible** — WCAG-compliant contrast, keyboard navigation, `prefers-reduced-motion` support
- 📝 **Contact Forms** — Validated inquiry and manpower request forms
- 🗺️ **Google Maps** — Embedded office location map

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- A local server for development (optional)

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Tharumika/deshakthi-website.git

# Navigate to the project
cd deshakthi-website

# Option 1: Open directly
# Just open index.html in your browser

# Option 2: Use a local server (recommended)
python -m http.server 8080
# Then visit http://localhost:8080
```
